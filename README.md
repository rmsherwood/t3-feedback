# T3 Live — Product Feedback & Prototypes

Working HTML prototypes and structured product feedback prepared after participating in the Momentum Express Virtual Trading Floor® open house (June 2026).

## Pages  

| File | Description |
|------|-------------|
| `index.html` | Feedback overview — priorities, concepts, growth levers |
| `t3-room.html` | Redesigned VTF® chat interface (Discord/Slack conventions) |
| `t3-offers.html` | Redesigned offers & pricing page (side-by-side comparison table) |
| `t3-login.html` | Redesigned room entry & login page |

## Viewing

Open `index.html` in a browser. The "Open prototype" buttons on the feedback page open each prototype directly in a new tab — no server required.

Live at: https://rmsherwood.github.io/t3-feedback/

## Notes

- All prototypes are standalone HTML files with no external dependencies beyond Google Fonts
- Prototypes are illustrative — they demonstrate layout, UX patterns, and content hierarchy, not production-ready code
- Feedback reflects one participant's experience during the June 2026 open house week
